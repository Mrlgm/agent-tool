# 🎨 图片生成前端代码 - 第四期需求文档

## 项目名称
**Image2Code** - 图片转前端代码生成平台

## 阶段目标
在前三期基础上，增加高级功能包括项目管理、历史记录、多图处理、性能优化、安全增强等，打造完善的产品级系统。

---

## 一、本期新增功能概览

### 1.1 功能清单

| 序号 | 功能模块 | 功能描述 | 优先级 |
|------|----------|----------|--------|
| 1 | 多图上传 | 支持多张图片同时上传 | P0 |
| 2 | 项目导出 | 导出完整项目ZIP包 | P0 |
| 3 | 历史记录 | 查看和管理历史生成记录 | P0 |
| 4 | 会话管理 | 支持多个项目会话 | P1 |

- ✅ 图片上传与分析

### 2.2 项目导出模块

#### 功能需求
- [ ] 导出为ZIP压缩包
- [ ] 包含完整项目结构
- [ ] 包含配置文件
- [ ] 包含依赖说明（README.md）
- [ ] 支持自定义项目名称
- [ ] 导出进度显示
- [ ] 大项目分片导出

#### 导出内容示例
```
project-name.zip
├── src/
│   ├── components/
│   ├── pages/
│   ├── styles/
│   └── ...
├── public/
├── package.json
├── tsconfig.json
├── vite.config.ts
├── .env.example
├── README.md          # 项目说明
└── INSTALL.md         # 安装说明
```

#### API设计
```typescript
// 导出请求
POST /api/project/export
{
  taskId: "xxx",
  projectName: "my-login-page"
}

// 响应
{
  downloadUrl: "/downloads/xxx.zip",
  expiresAt: "2024-01-01T00:00:00Z"
}
```

### 3.3 数据库设计（可选）

```sql
-- 用户表
CREATE TABLE users (
  id VARCHAR(36) PRIMARY KEY,
  email VARCHAR(255) UNIQUE,
  password_hash VARCHAR(255),
  created_at TIMESTAMP DEFAULT NOW()
);

-- 项目表
CREATE TABLE projects (
  id VARCHAR(36) PRIMARY KEY,
  user_id VARCHAR(36),
  name VARCHAR(255),
  framework VARCHAR(50),
  status VARCHAR(20),
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);

-- 历史记录表
CREATE TABLE history (
  id VARCHAR(36) PRIMARY KEY,
  project_id VARCHAR(36),
  images JSONB,
  code JSONB,
  version INT,
  created_at TIMESTAMP DEFAULT NOW()
);
```
